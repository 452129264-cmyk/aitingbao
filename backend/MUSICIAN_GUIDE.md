## 1. 腾讯音乐人（QQ音乐/酷狗/酷我）

### 平台特点
- **一次入驻，三平台分发**：QQ音乐、酷狗音乐、酷我音乐
- **API支持**：完整的开放API
- **审核周期**：1-3个工作日

### 入驻条件
1. 需上传至少3首原创作品
2. 持有作品版权或获得授权
3. 账号需完成实名认证

### 入驻流程

#### 第一步：访问入驻页面
```
https://y.tencentmusic.com/artist/apply
```

#### 第二步：填写资料
- 选择艺人类型（个人/机构）
- 填写基本信息
- 上传代表作品

#### 第三步：等待审核
- 审核时间：1-3个工作日
- 审核结果会以短信/邮件通知

### API申请

#### 1. 访问开放平台
```
https://y.tencentmusic.com/document/docs
```

#### 2. 创建应用
- 登录腾讯音乐人后台
- 进入「开放平台」→「我的应用」
- 点击「创建应用」

#### 3. 获取凭证
创建成功后获得：
- **App ID**: 应用唯一标识
- **App Key**: 应用密钥
- **Token**: 访问令牌（需OAuth获取）

#### 4. 配置凭证
```javascript
{
  platformId: 'qqmusic',
  credentials: {
    appId: 'YOUR_APP_ID',
    appKey: 'YOUR_APP_KEY',
    token: 'YOUR_ACCESS_TOKEN',
    artistId: 'YOUR_ARTIST_ID'
  }
}
```

### API接口权限
| 接口 | 权限 |
|------|------|
| 上传音频 | 音乐人权限 |
| 查询审核状态 | 音乐人权限 |
| 获取作品信息 | 基础权限 |

## 2. 网易音乐人

### 平台特点
- **优质社区**：用户活跃度高
- **个性化推荐**：算法推荐精准
- **API支持**：EAPI加密协议

### 入驻条件
1. 需上传至少1首原创作品
2. 作品需符合平台调性
3. 账号需完成实名认证

### 入驻流程

#### 第一步：访问入驻页面
```
https://music.163.com/st/musician
```

#### 第二步：申请成为音乐人
- 使用网易云音乐App扫码
- 填写音乐人资料
- 上传代表作品

#### 第三步：等待审核
- 审核时间：3-7个工作日
- 通过后获得音乐人后台访问权限

### API申请

#### 1. 访问开放平台
```
https://music.163.com/eapi/
```

#### 2. 获取凭证
联系网易云音乐商务获取：
- **App ID**: 应用标识
- **App Key**: 应用密钥
- **Device ID**: 设备标识（需生成）

#### 3. EAPI加密
网易云使用EAPI加密，需要实现：
```javascript
// EAPI加密流程
1. 构造请求参数
2. JSON序列化为字符串
3. 生成16位随机密钥
4. AES-128-CBC加密
5. 发送加密后的请求
```

#### 4. 配置凭证
```javascript
{
  platformId: 'netease',
  credentials: {
    appId: 'YOUR_APP_ID',
    appKey: 'YOUR_APP_KEY',
    token: 'YOUR_TOKEN',
    deviceId: 'YOUR_DEVICE_ID'
  }
}
```

## 3. 汽水音乐

### 平台特点
- **字节系产品**：与抖音深度联动
- **无公开API**：需浏览器自动化
- **推荐机制**：抖音算法加持

### 入驻条件
1. 需有抖音账号且完成实名认证
2. 需上传至少1首原创作品
3. 粉丝数无硬性要求

### 入驻流程

#### 第一步：访问创作平台
```
https://music.bytedance.com/creator
```

#### 第二步：登录/注册
- 使用抖音账号登录
- 完成创作者认证

#### 第三步：申请音乐人
- 进入「音乐人」模块
- 填写资料并上传作品
- 等待审核（1-3个工作日）

### Cookie获取

由于无公开API，需通过Cookie认证：

#### 1. 登录创作平台
访问 `https://music.bytedance.com/creator/login`

#### 2. 打开开发者工具
- 按F12打开DevTools
- 切换到「Network」标签

#### 3. 刷新页面
- 找到任意请求
- 查看「Request Headers」
- 复制「Cookie」字段

#### 4. 配置Cookie
```javascript
{
  platformId: 'shuishui',
  credentials: {
    cookies: 'sid_tt=xxx; sessionid=xxx; ...'
  }
}
```

### 注意事项
- Cookie有效期有限，需定期更新
- 建议设置自动刷新机制
- 遵守平台使用规范

## 4. 抖音音乐人

### 平台特点
- **巨大流量**：日活6亿+
- **短视频配乐**：歌曲爆发点
- **浏览器自动化**：无公开API

### 入驻条件
1. 抖音账号粉丝≥1000（建议）
2. 需上传至少1首原创作品
3. 完成实名认证

### 入驻流程

#### 第一步：访问创作服务平台
```
https://creator.douyin.com/creator-micro/home
```

#### 第二步：成为创作者
- 完成基础创作者认证
- 积累一定粉丝基础

#### 第三步：申请音乐人
- 进入「音乐人」模块
- 提交作品审核

### Cookie获取

#### 1. 登录创作平台
访问 `https://creator.douyin.com/`

#### 2. 获取Cookie
- F12打开DevTools
- Console执行：
```javascript
console.log(document.cookie)
```

#### 3. 获取关键Cookie
```
sessionid=xxx
csrf_token=xxx
```

#### 4. 配置
```javascript
{
  platformId: 'douyin',
  credentials: {
    cookies: 'sessionid=xxx; csrf_token=xxx'
  }
}
```

## 6. 小红书

### 平台特点
- **生活方式社区**：年轻用户集中
- **种草能力强**：容易病毒传播
- **无音乐人后台**：采用素材包方案

### 入驻条件
1. 小红书账号完成认证
2. 建议有内容发布记录
3. 无粉丝数要求

### 入驻流程

#### 第一步：访问创作者中心
```
https://creator.xiaohongshu.com/
```

#### 第二步：完成认证
- 完成实名认证
- 可选：完成专业号认证

#### 第三步：发布内容
- 直接在App中发布笔记
- 音频作为视频配乐

### 素材包使用

由于小红书无音乐人后台，我们提供素材包方案：

#### 1. 获取素材包
通过API生成分发任务：
```javascript
POST /api/distribute/submit
Body: {
  songId: 'xxx',
  platformIds: ['xiaohongshu']
}
Response: {
  success: true,
  data: {
    packageUrl: '/temp/package_xxx/',
    instructions: [
      '1. 打开小红书App...',
      '2. 点击"+"发布笔记...'
    ]
  }
}
```

#### 2. 素材包内容
```
package_xxx/
├── 音频文件.mp3          # 歌曲完整版
├── 封面图.jpg            # 建议尺寸1:1
├── 歌词文件.lrc          # 带时间轴
├── 歌曲信息.json         # 元数据
└── 上传指引.md          # 操作步骤
```

#### 3. 手动上传
按指引在小红书App中完成上传

## 常见问题

### Q1: 入驻审核被拒怎么办？
**A**: 检查作品是否符合平台调性，确保原创性，修改后重新提交。

### Q2: Cookie过期了怎么办？
**A**: 定期重新登录平台获取新Cookie，建议设置提醒。

### Q3: API配额用完了怎么办？
**A**: 联系平台申请提高配额，或升级开发者等级。

### Q4: 作品审核被下架怎么办？
**A**: 查看平台通知的违规原因，修改后重新提交。

### Q5: 多个平台同时分发有冲突吗？
**A**: 一般没有，但建议首发一个平台后再分发其他平台。
